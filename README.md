# Usage

1. Customize the variables `height` and `width` and the `defineColorValueOfPixel` function.
2. Compile.
3. Run `bitmapGenerator bitmapFileName` to save your bitmap in `bitmapFileName`. Default file name is `bitmap.bmp`.
